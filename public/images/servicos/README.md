# Imagens dos Serviços

Coloque as imagens dos serviços nesta pasta com os seguintes nomes:

1. **psicoterapia-individual.jpg** (ou .png) - Para Psicoterapia Individual
2. **terapia-casal.jpg** (ou .png) - Para Terapia de Casal
3. **terapia-familiar.jpg** (ou .png) - Para Terapia Familiar
4. **orientacao-profissional.jpg** (ou .png) - Para Orientação Profissional
5. **ansiedade-estresse.jpg** (ou .png) - Para Ansiedade e Estresse
6. **depressao.jpg** (ou .png) - Para Depressão

## Recomendações:
- Tamanho ideal: 400x300px ou proporção similar
- Formato: JPG ou PNG
- Peso: Otimizar para web (máximo 200KB por imagem)
- Estilo: Imagens profissionais, acolhedoras e relacionadas ao serviço

As imagens serão exibidas automaticamente nos cards de serviços.
